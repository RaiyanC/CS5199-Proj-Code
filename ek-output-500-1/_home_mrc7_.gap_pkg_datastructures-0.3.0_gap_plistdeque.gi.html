11# This file implements double ended queues (deques) using a circular buffer
12# stored in a GAP plain list.
13#
14# The four positions in a deque Q have the following purpose
15#
16# Q[1] - head, the index in Q[4] of the first element in the deque
17# Q[2] - tail, the index in Q[4] of the last element in the deque
18# Q[3] - capacity, the allocated capacity in the deque
19# Q[4] - factor by which storage is resized if capacity is exceeded
20# Q[5] - GAP plain list with storage for capacity many entries
21#
22# Global constants QHEAD, QTAIL, QCAPACITY, QFACTOR, and QDATA are bound to
23# reflect the above.
24#
25# When a push fills the deque, its capacity is resized by QFACTO R using
26# PlistDequeExpand. A new empty plist is allocated and all current entries of
27# the deque are copied into the new plist with the head entry at index 1.
28#
29# The deque is empty if and only if head = tail and the entry that head and tail
30# point to in the storage list is unbound.
